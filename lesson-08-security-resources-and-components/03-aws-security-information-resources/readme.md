## AWS Well-Architected Framework - Security Pillar

The Security Pillar provides guidance on protecting information, systems, and assets.

| Design Principle | Description |
|-----------------|-------------|
| Strong identity foundation | Implement least privilege, centralize identity |
| Enable traceability | Monitor, alert, audit all actions |
| Apply security at all layers | Edge, VPC, subnet, instance, OS, application |
| Automate security | Automated security mechanisms |
| Protect data in transit and at rest | Encryption, tokenization, access control |
| Keep people away from data | Reduce direct access to data |
| Prepare for security events | Incident response, simulations |
