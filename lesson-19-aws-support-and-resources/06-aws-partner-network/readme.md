## Introduction

The AWS Partner Network (APN) is a global community of partners that leverage AWS to build solutions and provide services to customers. Understanding the different partner types and how they can help organizations is relevant for the AWS Certified Cloud Practitioner exam, especially when discussing how to get help implementing AWS solutions.

## What is the AWS Partner Network?

The APN is AWS's partner program that includes:

- Technology companies building products on AWS
- Consulting firms helping customers implement AWS
- Managed service providers operating AWS environments
- Training and certification partners

## Partner Programs Comparison

| Aspect | Consulting Partner | Technology Partner |
|--------|-------------------|-------------------|
| **Primary Value** | Services & expertise | Products & software |
| **Delivery** | Professional services | Software deployment |
| **Engagement** | Project-based | License/subscription |
| **Specialization** | Industry/domain | Technology area |
| **Customer Relationship** | Hands-on implementation | Product support |

## ğŸ¯ Exam Tips

- **AWS Partner Network (APN)** includes both Consulting and Technology Partners
- **Consulting Partners** help customers implement and manage AWS solutions
- **Technology Partners** build software products that work with AWS
- **AWS Marketplace** is where you can find and purchase third-party software
- Know that partners are tiered: **Registered, Select, Advanced, Premier**
- Partners can have **Competencies** showing expertise in specific areas
- **AWS Marketplace** allows consolidated billing through your AWS account
- Partners help customers who lack in-house AWS expertise

## ğŸ’¡ Key Takeaways

1. **AWS Partner Network (APN)** is the global community of AWS partners
2. **Two main partner types**: Consulting Partners (services) and Technology Partners (products)
3. **Consulting Partners** help with migration, implementation, and managed services
4. **Technology Partners** build software solutions that run on or integrate with AWS
5. Partners are organized into **tiers**: Registered, Select, Advanced, Premier
6. **Competencies** indicate deep expertise in specific areas like Migration or Security
7. **AWS Marketplace** provides easy procurement with consolidated AWS billing
8. Consider engaging partners when you need expertise, speed, or specialized solutions
